# FileManager.API

A clean ASP.NET Core Web API for handling file operations like uploading, downloading, and streaming files.

---

## Features

- **Upload single file**  
  Upload a single file to the server and get its unique ID.

- **Upload multiple files**  
  Upload multiple files at once and get their IDs.

- **Upload image**  
  Upload an image file.

- **Download file by ID**  
  Download a file using its GUID.

- **Stream file by ID**  
  Stream a file efficiently using FileStream, suitable for large files.

- **Cancellation support**  
  All methods support `CancellationToken` for better control.

---
